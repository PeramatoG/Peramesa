*** HELP ***
PERAMESA V3.0

*** TOP PANEL ***

Adjust channel levels and mutes for each send as needed.

Double-click the lower channel label to switch between:
    - Gray: this channel will NOT be sent to the console.
    - Orange: this channel WILL be sent to the console when requested.

When you change any channel property, the lower label turns orange to
indicate that this channel's values will be sent to the console.

*** BOTTOM PANEL ***

[SHOW NAME]     Edit the SHOW name here.
[CUE NAME]      Edit the current CUE name here.

[DESK HOST]     Enter the mixer HOST here. Double-click the field to edit
                it. If left empty, it returns to the default value.

[DESK PORT]     Enter the mixer PORT here. Double-click the field to edit
                it. If left empty, it returns to the default value.

[OSC HOST]      Enter the HOST for OSC here. Double-click the field to edit
                it. If left empty, it returns to the default value.

[OSC PORT]      Enter the PORT for OSC here. Double-click the field to edit
                it. If left empty, it returns to the default value.


[CUE LIST]      Jump to a specific CUE by clicking it in the list.
[<< PREV]       Jump to the previous CUE.
[NEXT >>]       Jump to the next CUE.
[MOVE UP]       Move the selected CUE one position up.
[MOVE DW]       Move the selected CUE one position down.
[NEW]           Create a new CUE at the end of the list.

[DELETE]        Delete the current CUE and advance to the next available CUE.

[AUTOSEND]      When checked, the program tries to send data to the console
                every time the CUE changes, as long as the connection is
                possible.
[SEND TO DESK]  Send the current top-panel values to the console.
[SELECT]        Mark every channel in the layer as modified. If you press
                [SEND TO DESK] afterwards, the program will try to send all
                channels in that layer to the console.
[UNSELECT]      Mark every channel in the layer as NOT modified. If you press
                [SEND TO DESK] afterwards, nothing in that layer will be sent.
[CLEAR]         Press once to unmute every channel and set them to level 0.
                Press twice to also unmark all channels for sending.
[GO ALL]        Unmute every channel in that layer without marking them for
                sending.
[MUTE ALL]      Mute every channel in that layer without marking them for
                sending.

[CMD Line]      Shows some of the actions performed by the console.

*** OSC ***

The program responds to the following OSC commands:

/go 0     Jump to the previous CUE
/go 1     Jump to the next CUE
/goto x   Jump to CUE x

To configure QLab:

Open [Settings] (gear icon bottom right) >> [Network] and configure a
patch with the same Host IP and the same port.

Create a [Network] event (target icon button), and in [Basics] select the
patch you created as [Destination] and [Type] >> [OSC message].
